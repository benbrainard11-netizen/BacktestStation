# SMT-conditioned FVG findings

_Generated by `scripts/smt_conditioned_fvg_analysis.py` on 2026-05-09T19:46:07.870070+00:00._

Compares FVG behavior in the **SMT-thesis-active window** vs the general FVG population.

**Active window definition.** For each SMT, the active window is `(smt_knowable_ts, period_N_close]`. Aligned FVG = same primary symbol, thesis-aligned side, knowable_ts inside the window. **Zero look-ahead** — FVGs whose c3 hadn't closed before period N's end are NOT counted.

**Comparison baseline.** General population = all FVGs of the same `(fvg_type, fvg_side, primary_symbol)` across the full history (includes both SMT-overlapping and non-SMT FVGs). This is a coarse baseline; a tighter comparison would be FVGs explicitly NOT in any SMT active window. This is queued for v2 if the coarse comparison shows signal.

**Metrics**:
- `tap_rate_pct` — % of FVGs whose wick entered the gap
- `close_inside_rate_pct` / `close_through_rate_pct` — close-based outcomes
- `pct_*_of_tapped` — among tapped, how the tap bar resolved
- `median_bars_to_tap` — typical lag from FVG formation to first tap
- `mean_post_tap_N_mfe` — average MFE in thesis direction measured FROM the tap bar's close, over N forward candles

**Caveat.** Descriptive only. Walk-forward OOS required before any of these become a strategy.
